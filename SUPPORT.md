# Support

## How to get help

### GitHub Discussions
For questions, ideas, or general discussion, open a [GitHub Discussion](https://github.com/Maless88/Roadboard/discussions).

This is the best place to ask:
- How does X work in RoadBoard?
- Is Y the right approach for my use case?
- What is the recommended way to configure Z?

### GitHub Issues
Use [GitHub Issues](https://github.com/Maless88/Roadboard/issues) for:
- Bug reports
- Feature requests
- Documentation improvements

Please use the provided issue templates. Do not open issues for general questions — use Discussions instead.

### Security vulnerabilities
Do **not** open a public issue for security vulnerabilities.
See [SECURITY.md](SECURITY.md) for the private reporting process.

---

## Maintenance status

RoadBoard 2.0 is actively developed. The `main` branch is the only supported branch.

Response times are best-effort. There are no SLA guarantees for community issues.
